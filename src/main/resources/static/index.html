Bem-vindo à API da plataforma Bunchin

API para gerenciamento de funcionários e controle de pontos.
Utilize os endpoints abaixo para acessar os dados.

  • GET /api/funcionario
  • GET /api/funcionario/{cpf}
  • POST /api/funcionario
  • POST /api/login
  • GET /api/ponto
  • POST /api/ponto
