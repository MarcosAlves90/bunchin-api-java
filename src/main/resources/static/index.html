Bem-vindo à API da plataforma Bunchin

API para gerenciamento de funcionários e controle de pontos.
Utilize os endpoints abaixo para acessar os dados.

  • Funcionário
  • GET /api/funcionario
  • GET /api/funcionario/{cpf}
  • GET /api/funcionario/busca?nome=substring
  • POST /api/funcionario
  • PUT /api/funcionario/{cpf}
  • DELETE /api/funcionario/{cpf}
  • Login
  • POST /api/login
  • Ponto
  • GET /api/ponto
  • GET /api/ponto/{id}
  • POST /api/ponto
  • PUT /api/ponto/{id}
  • DELETE /api/ponto/{id}
  • Reset de Senha
  • POST /api/storeResetCode
  • POST /api/checkEmailExists
  • POST /api/verifyResetCode
  • PUT /api/resetPassword
  • PUT /api/newPassword
